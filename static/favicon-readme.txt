# Favicon Setup

The radion theme expects a favicon at: `/static/favicon.ico`

To add your favicon:
1. Create a 16x16 or 32x32 pixel favicon.ico file
2. Place it in the static directory
3. The theme will automatically use it

You can generate a favicon from:
- https://favicon.io/
- https://realfavicongenerator.net/
- Or use any image editor to create a .ico file
